# Revolution Macro Guide and basic menu information

## version 0.8b (Aka. The Hourly Graph Update)

START - F1
PAUSE - F2
STOP - F3

# Tabs - 6 in total

- Gather Tab (1)
- Collect Tab (2)
- Planters Tab (3)
- Status Tab (4)
- Tools Tab (5)
- Settings Tab (6)

# Gather Tab (1)

## Fields

- To edit a field, click the pen icon, to paste pre-made settings, click the 6 dots icon, and click “Paste”. Find pre-made settings in the Discord server -> #patterns-configs
- Editing the Field (Suggested to copy and paste pre-made settings from the Revolution Discord
- Disable drift detection (uses supreme saturator to align character) -> click the pen icon -> click “align” under max backpack -> disable
- Boosts (Make sure sprinkler is in slot 1)

## General

- Set sprinkler to your current sprinkler in Bee Swarm
- If you are blue and macro in Pinetree, enable “Use Fast Pine Return Path”
- Keep everything else the same
- Conversion: Customizable to your liking
- Field Defaults: You don’t have to touch this

## AI Gather  Settings

- Enable “Use Full Model”, “Eco Mode” (Recommended) “Loot Mode” is optional
- Token Priority: Customizable to your liking

# Collect Tab (2)

## Dispensers

- Enable “Wealth Clock”, “Glue Dispenser” and “Blueberry/Strawberry Dispenser” Others are optional

## Bug Run

- If you have gifted vicious bee, set “Monster Respawn Percent” to “15%”

## Quests

- If you lack Polar Power, enable. Priority can stay “0”

## Buffs
- Sticker Stack: Optional  
- Mondo Chick: Recommended to enable Mondo Chick. Only enable the top one
- HQ Boosters: Enable your designated booster color. Recommended to gather in boosted field.

# Planters Tab (3)

- Duration (top left) Keep to “12” or “24” hours
- Clear Schedule - clears planter schedule (collect all planters before clearing)
- Refresh Schedule - refreshes planter schedule
- Settings - Recommended to enable “Gather Loot”, “Reset when Harvesting” Keep timeline mode to “Timer”

## Auto Planters

- Settings -> Enabled. Recommended to Harvest every “3” Hours. Set minimum and maximum to “3”
- Nectar Goals -> Find a nectar guide that correlates to the hours you macro on the discord server
- Fields -> Custom
- Planters -> Custom (Recommended to place comforting nectar in bamboo, because bees sip nectar when macroing in Pinetree

# Status Tab (4)

## Logs

- Used to help troubleshoot macro problems

## Webhook

How to connect your macro to a discord webhook (Recommended)

1. Make a new discord server
2. Edit/Press the gear icon on the “general” text channel (has gear icon on the right)
3. Once you open text channel settings, go to integrations Click webhooks, and make a new webhook
4. Copy the webhook url, and open Revolution Macro Paste the webhook url in the macro in the status tab>webhook
5. Add your discord ID for pings if something goes wrong (optional)
6. To do step 5, turn on developer mode on Discord, and copy your ID and post it into the macro under webhooks

- Enable everything in the “main” tab and disable everything in the “Searcher” tab. This is only if you aren’t using mains

# Settings Tab (6)

## Macro

- Appearance: Enable Dark Mode  
- Hotkeys: Toggle what starts the macro (ex. F1, F2, F3 - Ctrl Q, Crtl W, Crtl E,)
- Capture - If your screen is flashing white, change “Preferred Capture Mode” to “Legacy Monitor”
- Misc - Don’t touch if not necessary

## Game

- Movement -> Make sure Player Movespeed is correct (no haste activated). “Key Delay” should be “50”
- Routing -> Enable High Alignment if your FPS is lower than 60. Check FPS by pressing Shift + F5
- Join -> Set “Hive Claim Method” to “Detect”. You put your private server link in this category.
